Description:
This backend server application is intended
to be used as an API for a web application
that will be using this data in particular.

In order to run the program:
1. You need to have MySQL installed in your computer.
2. Must have the database configuration (will be added in files soon)
3. Must have installed in your computer, Eclipse, Maven.
4. Must have the MySQL server running.
5. Have the App.java running (the server app running).
6. Then begin to user the routes.
